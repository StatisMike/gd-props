Provided Methods§

fn register_loader()

Associated function to register the created [ResourceFormatLoader] in Godot’s ResourceLoader. To be used in ExtensionLibrary implementation in on_level_init() function, as shown in example below. Unregistering function GdPropLoader::unregister_loader should be used in conjuction, in on_level_deinit() function to prevent memory leaks.

§Example

use godot::init::*;
use gd_props::traits::GdPropLoader;

struct MyGdExtension;

unsafe impl ExtensionLibrary for MyGdExtension {
    fn on_level_init(_level: InitLevel) {
        if _level == InitLevel::Scene {
            MyResPluginLoader::register_loader();
        }   
    }
    fn on_level_deinit(deinit: InitLevel) {
        if deinit == InitLevel::Scene {
            MyResPluginLoader::unregister_loader();
        }
    }
}

fn unregister_loader()

Associated function to unregister the [ResourceFormatLoader] from Godot’s ResourceLoader. See details and example of usage in GdPropLoader::register_loader documentation.
